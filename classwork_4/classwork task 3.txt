Задача: Калькулятор заработной платы с учетом премий, налогов и переработок
Тебе нужно создать программу для расчета заработной платы сотрудника с учетом различных факторов, таких как базовая ставка, премии, переработка и налоги.

Пользователь вводит базовую ставку сотрудника (baseSalary). Если базовая ставка не указана (например, null или undefined), программа должна установить базовую ставку по умолчанию, равную 50000.

Пользователь вводит количество часов переработки (overtimeHours):

Если указано значение больше 40, программа должна вывести сообщение: "Чрезмерное количество часов переработки, устанавливаем 40".
Если значение не указано (например, null или undefined), оно должно быть установлено на 0.
Каждый час переработки оплачивается дополнительно в размере 1.5% от базовой ставки.
Пользователь вводит процент премии (bonusPercentage):

Если процент премии больше 50%, программа должна вывести сообщение: "Премия слишком велика, устанавливаем 50%".
Если процент премии не указан, он должен быть установлен на 10%.
Премия рассчитывается как процент от базовой ставки.
Пользователь вводит, удерживать ли налог (applyTax):

Если applyTax равен true, программа должна удержать 13% от итоговой зарплаты.
Если applyTax равен false или не указан, налог не удерживается.
Программа должна рассчитать и вывести итоговую зарплату с учетом всех условий.

Подсказки:
Используй оператор нулевого слияния для значений по умолчанию.
Применяй условные операторы и операторы сравнения для проверки данных.
Используй логические операторы для комбинирования условий.

Пример работы программы:

Ввод:
let baseSalary = null;
let overtimeHours = 50;
let bonusPercentage = 60;
let applyTax = true;

Вывод:
"Чрезмерное количество часов переработки, устанавливаем 40"
"Премия слишком велика, устанавливаем 50%"
"Итоговая зарплата: 67500"

Ввод:
let baseSalary = 80000;
let overtimeHours = 20;
let bonusPercentage = 15;
let applyTax = false;

Вывод:
"Итоговая зарплата: 98200"
