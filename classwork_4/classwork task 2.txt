Задача: Программа расчета стоимости подписки с учетом скидок и акций
Тебе нужно создать программу для расчета стоимости подписки на сервис с учетом различных акций, скидок и условий.

Пользователь вводит базовую стоимость подписки (basePrice). Если базовая стоимость не указана (например, null или undefined), программа должна установить цену по умолчанию, равную 500.

Пользователь вводит тип подписки (subscriptionType):

Возможные значения: "premium", "standard", "basic".
Если тип подписки "premium", цена увеличивается на 50%.
Если тип подписки "standard", цена остается прежней.
Если тип подписки не указан или указано неверное значение, программа должна установить тип подписки как "basic" и уменьшить цену на 20%.
Пользователь вводит промокод (promoCode):

Если промокод "SALE20", применяется скидка 20% от текущей стоимости.
Если промокод "VIP", применяется скидка 30% от текущей стоимости.
Если промокод указан неверно или не указан, цена не меняется.
Пользователь указывает, применить ли дополнительные налоги (applyTax):

Если значение applyTax равно true, программа должна добавить к цене налог в 15%.
Если значение applyTax равно false или не указано, налог не добавляется.
Программа должна рассчитать и вывести финальную стоимость подписки с учетом всех условий.

Подсказки:
Используй оператор нулевого слияния для значений по умолчанию.
Применяй условное ветвление и операторы сравнения для обработки условий.
Используй логические операторы для комбинирования условий.
Пример работы программы:

Ввод:
let basePrice = null;
let subscriptionType = "premium";
let promoCode = "SALE20";
let applyTax = true;

Вывод:
"Финальная стоимость подписки: 552"

Ввод:
let basePrice = 400;
let subscriptionType = "basic";
let promoCode = "VIP";
let applyTax = false;

Вывод:
"Финальная стоимость подписки: 224"
