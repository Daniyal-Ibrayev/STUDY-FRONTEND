Задача по теме: Циклы while и for
Программа расчета накоплений с учетом депозита, процентов и дополнительных взносов
Тебе нужно создать программу для расчета накоплений за несколько лет, учитывая начальный депозит, ежемесячные проценты и дополнительные ежегодные взносы.

Пользователь вводит начальный депозит (initialDeposit). Если депозит не указан (например, null или undefined), программа должна установить значение по умолчанию, равное 100000.

Пользователь вводит процентную ставку (monthlyInterestRate):

Это процент, который начисляется на депозит каждый месяц.
Если процентная ставка не указана (например, null или undefined), она должна быть установлена на 0.5%.
Если процентная ставка больше 2%, программа должна вывести сообщение "Процентная ставка слишком велика, устанавливаем 2%" и задать ее значение 2%.
Пользователь вводит сумму ежегодного взноса (annualContribution):

Если сумма не указана (например, null или undefined), программа должна установить ее значение на 0.
Сумма ежегодного взноса добавляется к накоплениям в конце каждого года.
Пользователь вводит срок накопления (years):

Если срок не указан, программа должна задать его по умолчанию на 5 лет.
Каждый год состоит из 12 месяцев, и процентная ставка применяется каждый месяц.
Программа должна рассчитать и вывести сумму накоплений на конец каждого года.

**Если накопления на каком-либо этапе превышают 1 000 000, программа должна вывести сообщение "Цель достигнута!" и завершить расчет, не выполняя его для оставшихся лет.

Подсказки:
Используй цикл for для итераций по годам и цикл while для начисления процентов в каждом месяце года.
Применяй операторы сравнения и условное ветвление для проверки условий.
Используй логические операторы для комбинирования условий, оператор нулевого слияния для значений по умолчанию.
Пример работы программы:

Ввод:
let initialDeposit = null;
let monthlyInterestRate = 2.5;
let annualContribution = 50000;
let years = 3;

Вывод:
"Процентная ставка слишком велика, устанавливаем 2%"
Год 1: Накоплено 686280
Год 2: Накоплено 1337096
"Цель достигнута!"

Ввод:
let initialDeposit = 200000;
let monthlyInterestRate = 1;
let annualContribution = 30000;
let years = 2;

Вывод:
Год 1: Накоплено 268364
Год 2: Накоплено 342091
